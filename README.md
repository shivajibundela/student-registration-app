# 🎓 Student Registration System

A beginner-friendly web app built using **Streamlit** in Python 
that allows students to register and get instant admission analysis.

## 🚀 Features
- 📝 Enter student name, age, and course
- 📌 Real-time input summary
- 📊 Age-based admission analysis
- 🎨 Clean and interactive UI

## 🛠️ Tech Stack
- Python 3
- Streamlit

## ▶️ How to Run

git clone https://github.com/yourusername/student-registration
cd student-registration
pip install streamlit
streamlit run app.py
